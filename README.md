# NestJS REST API for Digital Wallet Dashboard App

CRUD REST API using the NestJS framework and MongoDB/Mongoose.

## Setup

Create a `.env` file in the root of your project:

```dosini
mongoURI=YOURMONGOURI
```

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
